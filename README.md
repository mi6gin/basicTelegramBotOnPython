# Шаблон для телеграм-бота на Aiogram

Это базовый шаблон для создания телеграм-бота с использованием библиотеки `aiogram`. Проект имеет модульную структуру для лёгкого расширения.

## Структура проекта

-   `main.py`: Основная точка входа для запуска бота.
-   `src/`: Папка с исходным кодом.
    -   `app.py`: Настройка и запуск бота, подключение обработчиков и middleware.
    -   `settings.py`: Конфигурация бота, загрузка данных из `.env` файла.
    -   `core/`: Ключевая логика (например, `notifier.py` для отправки уведомлений).
    -   `handlers/`: Обработчики сообщений и команд от пользователей.
    -   `middlewares/`: Промежуточные обработчики для сообщений.
    -   `utils/`: Вспомогательные утилиты (настройка логгирования, установка команд).
-   `logs/`: Папка для хранения лог-файлов.
-   `.env`: Файл для хранения секретных данных (не должен попадать в Git).
-   `requirements.txt`: Список зависимостей проекта.

## Настройка и запуск

1.  **Клонируйте репозиторий:**
    ```bash
    git clone <your-repo-link>
    cd <project-folder>
    ```

2.  **Создайте и активируйте виртуальное окружение:**
    ```bash
    python -m venv venv
    ```
    -   Windows: `venv\Scripts\activate`
    -   macOS/Linux: `source venv/bin/activate`

3.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Настройте переменные окружения:**

    Создайте файл `.env` в корневой папке проекта по примеру файла `.env.example` или просто создайте `.env` и добавьте в него следующие строки:

    ```env
    BOT_TOKEN="YOUR_TELEGRAM_BOT_TOKEN"
    ADMIN_ID="YOUR_TELEGRAM_USER_ID"
    ```

    -   `BOT_TOKEN`: Токен вашего бота, полученный от [@BotFather](https://t.me/BotFather).
    -   `ADMIN_ID`: Ваш Telegram User ID. Бот будет отправлять вам уведомление о запуске. Можно указать несколько ID через запятую (например, `12345,67890`).

5.  **Запустите бота:**
    ```bash
    python main.py
    ```

Бот начнёт работу и вы, как администратор, получите уведомление о его успешном запуске.
